# QC Requirements -- Pre-Submission Checklist

## 10-Point Audit
- [ ] Zero formula errors (#REF!, #DIV/0!, #VALUE!, #N/A, #NAME?)
- [ ] BOQ section totals = Cost Summary totals (exact match)
- [ ] No hardcoded values in formula cells (blue text = inputs only)
- [ ] All unit rates within +/-25% of published market range
- [ ] Quantities cross-checked against design dimensions (+/-5%)
- [ ] All productivity rates have cited source
- [ ] Markups match company policy for project phase
- [ ] Schedule critical path = resource-loaded duration (+/-10%)
- [ ] Risk register has minimum 8 risks identified
- [ ] All cost data has source reference with date

## Material Intensity Checks
| Parameter | Typical | Alarm |
|-----------|---------|-------|
| Rebar intensity | 80-120 kg/m3 concrete | <60 or >150 |
| Concrete / m3 stored | 0.15-0.25 m3/m3 | <0.10 or >0.35 |
| Formwork / m3 concrete | 4-8 m2/m3 | <3 or >10 |
| Preliminaries % | 8-15% of direct | <5% or >20% |
