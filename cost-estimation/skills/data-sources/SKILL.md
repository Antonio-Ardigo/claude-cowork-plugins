# Data Sources -- Audit Trail & References

## Purpose
Every number in an estimate must trace back to a published source with date, publisher, and URL. This skill provides the audit trail. Flag rates older than 12 months as potentially stale.

## Source Categories
1. **Pricing sources**: Material costs, labor rates, equipment rental
2. **Productivity sources**: Academic papers, contractor data, industry norms
3. **Standards sources**: Design codes, specifications, local regulations
4. **Market intelligence**: Growth forecasts, demand indicators, policy changes

## Verification Protocol
- Match queried data point to its source
- Provide: publisher, date, URL, coverage (countries)
- Note access type (free/paid)
- Flag if data is > 12 months old
- Recommend quarterly refresh for pricing, annual for benchmarks
