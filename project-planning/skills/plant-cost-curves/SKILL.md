# Plant Cost Curves -- Typical Configuration & Typical Project Duration

## How to Use These Curves

These curves represent **order-of-magnitude benchmarks** for typical plant configurations in a "standard" location (no extreme site conditions, developed infrastructure, moderate labor market). They are suitable for:
- Class 5 screening estimates (G0)
- Sanity-checking Class 4 estimates (G1)
- Identifying anomalies vs industry norms

Adjustments required for actual projects:
- **Location factor**: Labor rates, productivity, import duties, logistics
- **Currency/escalation**: Curves reference mid-2020s USD or EUR
- **Scope differences**: Battery limits, infrastructure, owner's costs
- **Market conditions**: Tight vs slack construction market
- **Site-specific**: Geotechnical, environmental, remote location premium

## Notes

- All costs are order-of-magnitude for screening and benchmarking. They include EPC scope but typically exclude land, financing, owner's costs, and permits.
- Duration ranges assume single-shift construction with moderate overlap of phases. Two-shift or accelerated schedules can reduce duration by 15-25% at higher cost.
- Scaling exponent (n) for capacity-factored estimates is technology-specific. Use 0.6-0.7 as default if no specific data available.
- Market conditions (tight vs slack) can cause +/-20% variation from these benchmarks.
- Always cross-reference with recent project data when available.
