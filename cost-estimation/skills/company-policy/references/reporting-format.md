# Mandatory Deliverables

| # | Deliverable | Format | Description |
|---|-----------|--------|------------|
| 1 | Cost Summary | XLSX sheet | 1-page with section totals |
| 2 | Detailed BOQ | XLSX sheet | 14 sections, item/unit/qty/rate/amount |
| 3 | Basis of Estimate | XLSX sheet | Key assumptions, rates, sources |
| 4 | Productivity Rates | XLSX sheet | Base rates + adjustment factors |
| 5 | Duration Calculation | XLSX sheet | Qty / (rate x gangs x factor) |
| 6 | Resource Histogram | XLSX sheet | Weekly headcount + chart |
| 7 | Sensitivity Analysis | XLSX sheet | Top-5 items +/-20%, tornado |
| 8 | Risk Register | XLSX sheet | Risks, probability, impact, mitigation |
| 9 | Cost Breakdown Charts | XLSX sheets | Pie, bar, area charts |
| 10 | Time Schedule / Gantt | XLSX sheet | Stacked bar Gantt chart |
| 11 | Engineering Sketches | XLSX sheet | SK-001 to SK-004 embedded PNGs |
| 12 | Executive Summary | DOCX | 5-7 page standalone document |
| 13 | Audit Report | DOCX | Formula audit + rate verification |

## Chart Requirements
- Pie: cost breakdown with category names + percentages
- Bar: by section, sorted by value
- Gantt: activity names on Y-axis, stacked horizontal bars
- Histogram: weekly headcount, peak highlighted
