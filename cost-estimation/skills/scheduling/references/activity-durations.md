# Activity Duration Calculations

## Formula
**Duration (days) = Quantity / (Gang output/day x Gangs x Productivity factor)**

## Example
Base slab rebar: 1,200t / (2.0 t/day x 5 gangs x 0.80) = 150 days

## Minimum Practical Durations
| Activity | Minimum | Reason |
|----------|---------|--------|
| Mobilization | 3-4 weeks | Camp, equipment delivery |
| Demobilization | 2 weeks | Equipment removal |
| Hydrostatic test (per cell) | 7 days | EN 1992-3 hold |
| Curing (water-retaining) | 7 days | EN 13670 minimum |
| Wall pour cycle (per lift) | 6-7 days | Form + rebar + pour + cure |
| Commissioning | 1-2 weeks | Valve, instrument testing |

## Multi-Cell Overlap
- 2 cells concurrent for walls (limited by formwork + crane)
- Stagger by 1-2 weeks between cell starts
- Base slab: sequential pour strips across all cells
