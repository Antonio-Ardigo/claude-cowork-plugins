# Connectors -- Concept Builder Plugin

This plugin is self-contained and works without external connections.

## Optional Integrations

| System | Purpose | Notes |
|--------|---------|-------|
| Web search | Look up recent research, verify historical facts | Enhances accuracy of historical context and current relevance |
| File system | Save/read concept .md files | Required for `/library` and file export |

## Currently Active

- **File system**: Concept files are saved to the current working directory as Markdown
