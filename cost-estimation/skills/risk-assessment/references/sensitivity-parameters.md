# Sensitivity Analysis Parameters

## Parameters to Vary
| Parameter | Range | Sensitivity |
|-----------|-------|-------------|
| Concrete rate | +/-20% | HIGH (20-25% of direct cost) |
| Rebar rate | +/-20% | HIGH (25-35% of direct cost) |
| Formwork rate | +/-15% | MEDIUM (8-12%) |
| Productivity factor | +/-25% | HIGH (affects duration + labor) |
| Duration | +/-20% | MEDIUM (affects preliminaries) |
| Contingency % | +/-5% | MEDIUM |
| OH&P % | +/-3% | LOW-MEDIUM |

## Top-5 Cost Drivers (Typical RC Tank)
1. Reinforcement steel (25-35%)
2. Concrete (20-25%)
3. Preliminaries/site staff (10-15%)
4. Formwork (8-12%)
5. Waterproofing (5-8%)

## Tornado Diagram Format
- Horizontal bar chart, sorted by impact (widest at top)
- Center line = base case total
- Each bar: low to high cost for that parameter

## Break-Even Analysis
- Rebar: project unviable typically > $1,800/t
- Productivity: schedule exceeds contract < 0.60x factor
- Concrete: exceeds budget typically > $250/m3
