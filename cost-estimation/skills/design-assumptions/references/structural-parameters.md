# Structural Parameters

## Concrete Grades
| Element | Grade | fck (MPa) | Application |
|---------|-------|-----------|-------------|
| Blinding | C12/15 | 12 | Non-structural leveling |
| Roof slab, internal walls | C30/37 | 30 | Standard structural |
| Walls, base slab | C35/45 | 35 | Water-retaining |
| Columns, beams (optional) | C40/50 | 40 | High-strength elements |

## Reinforcement
- Grade: B500B (fy = 500 MPa), ductility class B
- Seismic zones: B500C (ductility class C)
- Bar sizes: 12, 16, 20, 25, 32mm standard

## Cover Requirements
| Face | Cover (mm) | Reason |
|------|-----------|--------|
| Water face | 50 | Exposure + crack width |
| Earth face | 40 | Buried concrete |
| Internal (dry) | 35 | Standard |
| Roof top | 25 | Protected by screed |
| Blinding contact | 75 | Uneven surface |

## Crack Width Limits
| Class | Max Crack Width | Application |
|-------|----------------|-------------|
| Tightness Class 1 | 0.2mm | Water-retaining (EN 1992-3) |
| Tightness Class 0 | 0.3mm | Non-retaining elements |

## Design Life & Durability
- Design life: 50 years minimum (EN 1990 Class 4)
- Exposure: XC2/XC3 (carbonation), XS1 (chloride, if coastal)
- Rebar intensity: 80-120 kg/m3 concrete (typical water tanks)
