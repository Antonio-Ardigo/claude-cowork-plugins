# Dewatering Methods

## Sump Pumping (minor seepage)
- Collect in sump pits at corners
- Submersible pumps: 5-10 l/s each
- Cost: $500-800/day
- Suitable for: perched water, surface runoff

## Wellpoint System (shallow, <5m)
- Header pipe + wellpoints at 1-2m spacing
- Suction limit: 5m (atmospheric)
- For sand/gravel with high permeability
- Installation: 1-2 weeks
- Running cost: $1,500-2,500/day

## Deep Wells (>5m)
- Drilled wells (150-300mm dia) with submersible pumps
- Spacing: 10-30m depending on permeability
- Installation: 2-4 weeks
- Running cost: $2,000-4,000/day

## Saudi/Desert Context
- Inland: generally low water table (>5m)
- Coastal (Jeddah, Dammam): may encounter high water table
- Sabkha soils: saturated at shallow depth, aggressive to concrete
- Budget allowance: $35,000 LS (conservative for desert sites)

## Monitoring
- Piezometers: install before excavation, read daily
- Settlement: adjacent structures within 2x excavation depth
- Discharge quality: check suspended solids before release
