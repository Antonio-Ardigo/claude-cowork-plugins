## Currently Active

- **docx-js**: Node.js -- professional DOCX with tables, headers/footers
- **matplotlib**: Python -- engineering sketches (plan, section, detail, roof)
- **LibreOffice**: headless formula recalculation via recalc.py

## Setting Up Connectors
Configure MCP servers in `.mcp.json` for external tool integration.
