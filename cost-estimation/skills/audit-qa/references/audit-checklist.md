# 10-Point Estimate Audit Checklist

| # | Check | Method | Pass Criteria |
|---|-------|--------|---------------|
| 1 | Formula errors | Recalculate (LibreOffice) | Zero errors |
| 2 | BOQ-Summary match | Compare section totals | Exact match |
| 3 | No hardcoded formulas | Inspect formula cells | All use cell refs |
| 4 | Rate reasonableness | vs market-*.md ranges | Within +/-25% |
| 5 | Quantity verification | Recalculate 3-5 items | Within +/-5% |
| 6 | Productivity sourced | Check each rate | All have source + date |
| 7 | Markup compliance | vs company policy | Matches phase |
| 8 | Schedule-duration match | Compare totals | Within +/-10% |
| 9 | Risk register complete | Count risks | Min 8 with mitigation |
| 10 | Source references | Check provenance | Publisher + date + URL |

## Audit Report Sections
1. Scope of Audit
2. Pricing Verification (market vs model, source)
3. Formula Audit (10 checks, PASS/FAIL)
4. Workbook Structure (sheet count, naming)
5. Cost Reasonableness ($/m3 vs benchmark)
6. Location-Specific Observations
7. Recommendations
8. Conclusion (VERIFIED / REQUIRES CORRECTION)
