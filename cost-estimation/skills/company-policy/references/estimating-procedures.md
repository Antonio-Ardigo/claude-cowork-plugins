# Estimating Procedures

## Step-by-Step Workflow
1. **Receive scope**: Client brief, drawings, specifications, site visit report
2. **Quantity takeoff**: Measure from drawings using measurement-rules
3. **Apply unit rates**: Select market file, apply base rates to quantities
4. **Adjust productivity**: Apply climate/location factors to duration calculation
5. **Specialist quotes**: Obtain subcontractor quotations for M&E, instrumentation
6. **Build schedule**: CPM network, resource histogram, Gantt chart
7. **Risk assessment**: Risk register, sensitivity analysis on top-5 cost items
8. **Internal review**: Peer review by senior estimator, check against benchmarks
9. **Management markup**: Apply contingency, escalation, OH&P per approval authority
10. **Final deliverables**: XLSX workbook + Executive Summary DOCX + Audit Report DOCX

## File Naming Convention
`[Asset]_[Volume]_[Market].xlsx` (e.g., `Tank_100000m3_SAUDI.xlsx`)
Revisions: append `_REV01`, `_REV02`

## Version Control
- Save each revision as separate file (do not overwrite)
- Record revision date, author, and change description
- Final issue: remove "DRAFT" watermark
