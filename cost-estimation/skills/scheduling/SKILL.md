# Scheduling -- Time & Resource Planning

## Purpose
Scheduling converts quantities + productivity rates into a time schedule with resource histograms. Duration = Quantity / (Daily rate x Number of gangs x Climate factor). Dependencies define the critical path.

## Key Outputs
- Activity list with durations, start/finish dates
- Gantt chart (stacked bar format)
- Resource histogram (weekly headcount)
- Critical path identification
- S-curve (cumulative progress)
