# RC Rectangular Water Tank -- Design Parameters

## 100,000 m3 Reference Design (2x2 Cells)

### Geometry
| Parameter | Value | Notes |
|-----------|-------|-------|
| Total volume | 100,000 m3 | 4 cells x 25,000 m3 |
| Cell arrangement | 2 x 2 | Shared internal walls |
| Cell dimensions | 65m x 65m x H=6.5m | Per cell |
| Total footprint | 130.4m x 130.4m | Including all walls |
| Water depth | 5.92m | Operating level |
| Freeboard | 0.58m | To top of wall |

### Structural Elements
| Element | Dimension | Concrete Grade |
|---------|-----------|---------------|
| External walls | 400mm thick | C35/45 |
| Internal walls | 300mm thick | C30/37 |
| Base slab | 400mm thick | C35/45 |
| Blinding | 75mm thick | C12/15 |
| Roof slab | 200mm thick | C30/37 |
| Roof beams | 300mm x 600mm | C30/37 |
| Columns | 400mm x 400mm | C35/45 |
| Kicker | 150mm height | Same as wall |

### Column Grid
- Spacing: 7.0m center-to-center both directions
- Grid: 8 x 8 = 64 columns per cell
- Total: 256 columns (4 cells)
- Beam grid follows column grid

### Construction Parameters
- Excavation depth: 1.0m below FGL
- Base slab pour strips: max 30m x 15m
- Wall lifts: 2 lifts @ 3.25m each
- Working space: 1.0m each side minimum
